# Assets Folder

## Images

Place your logo image here as `logo.png` for the login screen.

The login screen will display:
- Your logo on the left panel (desktop/tablet)
- A fallback store icon if logo.png is not found

Recommended logo specifications:
- Format: PNG with transparent background
- Size: 200x200 pixels (or larger, will be scaled)
- Style: Should work well on red gradient background
